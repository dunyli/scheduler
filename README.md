## Системный монитор и симулятор планировщика процессов
### Описание проекта
Данный проект представляет собой учебно-практическую утилиту для просмотра загруженности системы и симуляции алгоритмов планирования процессов. 

Программа сочетает в себе два режима работы:

1) Режим мониторинга системы (аналог top/htop) - отображает реальные процессы Windows,
загрузку CPU и использование памяти в реальном времени с интерактивным управлением.
2) Режим симулятора планировщика - демонстрирует работу алгоритмов планирования процессов (FIFO, Round Robin, Priority)
на виртуальных процессах с визуализацией и статистикой.

### Теоретическая часть
В проекте реализованы и исследованы следующие алгоритмы планирования:

1. FIFO (First In First Out)

Тип: Невытесняющий

Принцип: Процессы выполняются в порядке поступления

Применение: Пакетная обработка, очереди печати

Особенности: Простой, предсказуемый, но может вызывать "эффект конвоя"

2. Round Robin (Круговая очередь)

Тип: Вытесняющий

Принцип: Каждому процессу выделяется квант времени

Применение: Интерактивные системы, системы разделения времени

Особенности: Справедливое распределение CPU, частые переключения контекста

3. Priority Scheduling (Приоритетное планирование)

Тип: Вытесняющий (с приоритетным ускорением)

Принцип: Процессы с более высоким приоритетом выполняются первыми

Применение: Системы реального времени, мультимедиа
